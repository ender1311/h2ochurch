# H2O Church Columbus — Web App

The official web application for **H2O Church** in Columbus, Ohio.

## About

H2O Church is a community church in Columbus. This repository hosts the source
code for the church's web application — the front door for people to learn about
the church, find service times and locations, watch messages, connect with a
group, and get involved.

## Goals

- Help newcomers find service times, locations, and what to expect on a first visit.
- Make it easy to watch or listen to recent messages.
- Connect people to groups, serving teams, and events.
- Provide a simple way to give online.
